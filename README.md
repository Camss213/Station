# Fuel Finder

Application full-stack de comparaison des prix de carburant en France, pensée en mode PWA mobile-first.

## Structure

- `apps/web`: frontend React + Vite + Tailwind CSS
- `apps/api`: backend FastAPI + ingestion Open Data
- `docs`: schéma de base de données et notes d'architecture

## Stack retenue

- Frontend: React, Vite, Tailwind CSS
- Backend: Python, FastAPI, SQLAlchemy
- Base de données: PostgreSQL + PostGIS
- Cartographie: Leaflet
- Données: flux officiel prix des carburants
